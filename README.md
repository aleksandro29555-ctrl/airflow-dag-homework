# Что делает DAG

DAG загружает новые данные из CSV-файла в таблицу bd_shops.visits, затем вызывает SQL-функцию для обновления витрины bd_shops.daily_shop_visits_mart, после чего записывает в лог сообщение об успешном завершении.

## Логика метрики

В витрине рассчитываются:
	•	количество визитов в магазин за день;
	•	средний размер очереди;
	•	максимальный размер очереди.

Ключевая метрика — avg_line_size, которая позволяет оценивать нагрузку на магазин и находить точки с потенциальными проблемами обслуживания.

## Структура проекта
- dags/ — DAG Airflow
- sql/ — SQL-функция для витрины
- data/ — CSV с дополнительными данными
